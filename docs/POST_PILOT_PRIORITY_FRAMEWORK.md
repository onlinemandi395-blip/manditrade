# Post Pilot Priority Framework

Updated on 2026-06-02 to match the final live product model.

## Platform Shape

- 5 active roles
- 3 core commerce lanes plus specialized `Suta Mandi`
- no live client role
- admin-routed raw-material supply

## Priority Language

Use only these product terms in future planning:

- `Marketplace`
- `Marketplace Orders`
- `MandiPlace`
- `Mandi Orders`
- `Raw Materials`
- `Supply Orders`
- `Supply Requests`
- `Suta Mandi`
- `B2B Price`
- `Marketplace Price`
- `Mandi Price`

## Avoid In New Planning

- retired legacy buyer-role language
- removed private-network wording
- deprecated pre-cleanup order terminology
